# Sprint Plan

Rediscovering Faith is scheduled as four focused one-week sprints. Each sprint should end with a working, reviewable increment and a short recap commit or pull request summary.

## Sprint 1: Foundation & Core Experience

Goal: establish the core product experience and technical foundation.

Deliverables:

- Responsive app shell and visual direction
- Supabase project connection
- Authentication setup
- Podcast list and podcast detail flow
- Episode detail page
- Transcript display
- Initial database schema for users, podcasts, episodes, and transcripts

Portfolio evidence:

- Initial project setup commit
- README and sprint planning docs
- Screenshots of the first usable interface
- Sprint 1 recap in ClickUp and GitHub

Working files:

- [Sprint 1 backlog](sprint-1-backlog.md)
- [ClickUp import guide](clickup/import-guide.md)
- [ClickUp CSV import](clickup/sprint-1-clickup-import.csv)

## Sprint 2: Community & Engagement

Goal: turn passive listening into conversation and saved engagement.

Deliverables:

- Timestamped comments on episode transcripts
- Discussion boards and threaded replies
- Likes and reactions
- Saved collections and bookmarks
- Basic real-time updates where useful

Portfolio evidence:

- Feature commits grouped by user workflow
- Tests for comment and saved-item behavior
- Demo screenshots or short screen recording

## Sprint 3: Pastoral & Church Tools

Goal: support trusted leadership, moderation, and care workflows.

Deliverables:

- Pastor verification workflow
- Role-based access controls
- Moderation dashboard
- Prayer request wall
- Notifications and email alerts

Portfolio evidence:

- Role and permission documentation
- Moderation workflow demo
- Database policy notes

## Sprint 4: Admin, Groups & Deployment

Goal: prepare the application for a complete capstone presentation and portfolio handoff.

Deliverables:

- Group management
- Leader tools and resources
- Analytics dashboard
- Final responsive polish
- Production deployment
- Final README updates

Portfolio evidence:

- Deployment link
- Final screenshots
- Architecture notes
- Capstone completion summary
